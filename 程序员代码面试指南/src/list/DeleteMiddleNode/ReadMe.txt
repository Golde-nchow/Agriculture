给定链表的头结点，实习那删除链表的中间节点的函数
例如：
没有节点，不删除任何节点；
1->2，删除节点1；
1->2->3，删除节点2；
1->2->3->4，删除节点2；
1->2->3->4->5，删除节点3；

--------------------------------------------

书本思路：
1、如果是只有一个节点，直接返回
2、如果是2个节点（if cur.next.next == null），直接返回cur.next.next
3、如果是3、4个节点，删除的肯定是2
4、如果是5、6个节点，删除的肯定是3
5、以此类推
6、首先指针去到第三个节点，while (cur.next && cur.next.next为空)，
   pre = pre.next,
   cur = cur.next.next
   保证3、4不移动，保证5、6只移动一次，保证7，8移动2次
7、移动完之后，pre.next = pre.next.next;
8、返回节点
