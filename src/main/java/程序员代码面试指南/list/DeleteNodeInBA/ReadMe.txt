给定链表的头结点head、整数a和b，实现删除位于a/b处节点的函数.
例如：
链表：1->2->3->4->5，假设a/b的值为r
如果r等于0，不删除任何节点;
如果r在(0,1/5]上，删除节点1；
如果r在(1/5,2/5]上，删除节点2；
如果r在(2/5,3/5]上，删除节点3；
如果r在(3/5,4/5]上，删除节点4；
如果r在(4/5,1]上，删除节点5；
如果r大于1，不删除任何节点.

书本思路：
1、计算出链表的个数，然后根据b/a的比例，再乘以链表个数num，得出范围（0，b/a * num ]
   其实跟上面是对等的，a/b = r --> a/b * num = r * num;

2、根据上面的计算和题目的要求：计算最接近正无穷大的浮点数，所以会使用Math.ceil()
   如果r在(0,1]上，删除节点1；
   如果r在(1,2]上，删除节点2；
   如果r在(2,3]上，删除节点3；
   如果r在(3,4]上，删除节点4；
   如果r在(4,5]上，删除节点5；
   如果r大于5，不删除任何节点.
